The design of this site was built iteratively based on your input and common patterns for community organization websites. Here's how the decisions were made:

Layout Structure:

Sticky header with navigation — standard for community sites so people can always get to events quickly
Hero sections on each page — big visual headers with the key info (event name, date, location) front and center
Card-based event listing — grid of cards on the events page, each showing image, title, date, location. This is the most scannable format for browsing multiple events
Single-column event detail pages — long scrolling pages with sections stacked vertically: hero → event info → ticket purchase → sponsors → vendor registration → contact. This works well on mobile where most of your visitors probably are
Design System:

shadcn/ui components — pre-built, accessible React components (buttons, cards, forms, dialogs, tabs). These follow modern design conventions and look clean without heavy customization
Tailwind CSS — utility-based styling so every element is consistent in spacing, colors, and typography
HSL color variables — your primary color, backgrounds, borders, text colors all defined as CSS variables so changing the theme is just changing a few values
Lucide icons — clean, consistent icon set throughout
Key Design Choices:

Mobile-first — hamburger menu on small screens, full nav bar on desktop
Category tabs on events page — lets people filter by event type without leaving the page
Sponsor logos grouped by tier — Presenting sponsors biggest, then Gold, Silver, Supporting in descending visual prominence
Inline ticket purchase form — no separate checkout page, the form is right on the event page so there's minimal friction between deciding to buy and entering info
Most of these decisions were made by me based on what works best for this type of site. You gave direction on specific things — what events to add, what content to show, what features to turn on — and I made the layout and design decisions to present that content effectively. If there's anything about the current design you want changed, just let me know.
